## ? 디자인 개선 제안

### 1. 그래프 시각화 강화
#### 현재 상태
- 단색 검은 배경 (#0a0a0a)
- 기본 노드 스타일 (원형/사각형)
- 단순 링크 화살표

#### 개선 방안
? **그리드 패턴 배경**
```css
.graph-background {
  background-image: 
    linear-gradient(rgba(255,255,255,0.02) 1px, transparent 1px),
    linear-gradient(90deg, rgba(255,255,255,0.02) 1px, transparent 1px);
  background-size: 50px 50px;
}
```

? **미니맵 추가** (우상단)
- 전체 그래프 개요 보기
- 현재 뷰포트 위치 표시
- 클릭으로 빠른 이동

? **줌 컨트롤러**
- 좌하단 + / - 버튼
- 줌 레벨 퍼센트 표시 (50% ~ 200%)
- Fit to Screen 버튼

? **노드 시각화 개선**
- 그룹별 그라데이션 배경
- 호버 시 부드러운 확대 효과
- 연결선 개수 표시 (작은 뱃지)
- 최근 수정 시간 표시 (아이콘)

### 2. 노트 패널 UX 개선
#### 현재 상태
- 우측 슬라이드 패널
- Summary + Detailed Note 구분
- 자동 저장

#### 개선 방안
? **탭 시스템 확장**
```
? Note | ?? Tags | ? Files | ? Links | ? Stats
```

1?? **Tags 탭**
- 태그 입력 및 관리
- 태그 클릭 시 필터링
- 태그별 색상 설정
- 자동 완성 기능

2?? **Files 탭**
- PDF/이미지 첨부
- 드래그 앤 드롭 업로드
- 미리보기 기능
- 파일 크기/날짜 표시

3?? **Links 탭**
- 관련 논문 URL 관리
- DOI 자동 메타데이터 가져오기
- 링크 미리보기 (Open Graph)
- 외부/내부 링크 구분

4?? **Stats 탭**
- 작성 시간 추적
- 수정 히스토리
- 워드 카운트 그래프
- 참조 횟수 통계

? **마크다운 지원**
- 실시간 미리보기 (Split View)
- 툴바: Bold, Italic, Code, Link
- 단축키: Ctrl+B, Ctrl+I, Ctrl+K
- 코드 블록 하이라이팅

? **버전 관리**
- 자동 스냅샷 (30분마다)
- 타임라인 뷰
- 이전 버전 복원
- Diff 비교 보기

### 3. 인터랙션 개선
#### 현재 상태
- 좌클릭: 토글 메뉴
- 우클릭: 컨텍스트 메뉴
- 드래그: 노드 이동

#### 개선 방안
? **멀티 선택**
- Shift + Click: 여러 노드 선택
- Ctrl + A: 전체 선택
- 선택 영역 드래그 (Marquee Selection)
- 일괄 스타일 적용

? **노드 그룹핑**
- 선택한 노드들을 그룹으로 묶기
- 그룹명 설정 (예: "AI 관련 논문")
- 그룹 배경색 설정
- 그룹 접기/펼치기

? **링크 인터랙션**
- 링크 호버 시 툴팁 (관계 설명)
- 링크 클릭 시 상세 정보
- 링크 타입 변경 (forward/backward/related)
- 링크 강도/중요도 표시 (선 굵기)

? **스마트 배치**
- 자동 정렬 (Tree/Force/Grid)
- 중요도에 따른 크기 조절
- 관련도 높은 노드 자동 그룹핑
- 애니메이션 전환

---

### 4. 검색 & 필터링
#### 개선 방안
? **전역 검색 바** (상단 중앙)
- 제목/내용/태그 통합 검색
- 실시간 검색 결과 (드롭다운)
- 검색 히스토리
- 정규식 지원

? **고급 필터**
```
? 날짜: 최근 7일 / 30일 / 전체
?? 태그: AI, ML, CV (다중 선택)
? 그룹: Core / Forward / Backward
? 상태: 읽음 / 읽지않음 / 중요
```

? **스마트 뷰**
- "최근 수정" 뷰
- "읽지 않은 논문" 뷰
- "중요 표시" 뷰
- "고립된 노드" 뷰 (연결 없음)

### 5. 색상 테마
#### 현재 상태
- 다크 테마 고정 (#0a0a0a 배경)

#### 개선 방안
? **테마 시스템**
```javascript
// Dark Theme (현재)
background: #0a0a0a
panel: #0f0f10
accent: #14b8a6 (teal)

// Light Theme
background: #f8fafc
panel: #ffffff
accent: #0891b2

// Solarized
background: #002b36
panel: #073642
accent: #268bd2

// Nord
background: #2e3440
panel: #3b4252
accent: #88c0d0
```

? **다크/라이트 토글**
- 우상단 테마 스위치
- 시스템 설정 따르기
- 부드러운 전환 애니메이션
- 각 테마별 최적화된 색상

### 3. 생산성 도구
? **단축키**
```
Ctrl+N: 새 노드 추가
Ctrl+F: 검색
Ctrl+S: 수동 저장
Delete: 노드 삭제
Ctrl+Z/Y: 실행 취소/다시 실행
Space: 그래프 중앙 정렬
```

? **퀵 액션**
- 노드에서 노드로 드래그하여 링크 생성
- Alt + 클릭으로 빠른 노트 작성
- Ctrl + D로 노드 복제
- / 키로 커맨드 팔레트 열기
